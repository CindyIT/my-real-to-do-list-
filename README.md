
# Cindy's To-Do List

## Overview

This project is a simple and interactive To-Do List web application where users can add, mark as completed, and remove tasks. It is designed to help users manage and keep track of their daily tasks with a clean and intuitive user interface. The app is built using HTML, CSS, and JavaScript.

### Features:
- **Add Tasks**: Users can input tasks and press "Enter" to add them to the list.
- **Mark as Completed**: Left-click on a task to mark it as completed.
- **Remove Tasks**: Right-click on a task to remove it from the list.
- **Responsive Design**: The app is responsive and works across different screen sizes.
